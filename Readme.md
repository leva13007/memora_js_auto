# Memora UI Test (JavaScript + Playwright) Project


Before you start the project, you need to create a .env file in the root of the project and add the following variables. 
To create it, run the following command in the terminal
```bash
cp .env.example .env
```

Then, open the .env file and add the following variable:

```bash
BASE_URL=http://localhost:3000
```
```bash
or for Storybook from the Stream:
HOST_URL=https://leva13007.github.io/memora-cards-storybook/
```
